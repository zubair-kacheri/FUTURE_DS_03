#  Student Satisfaction Survey Analysis

##  Project Overview

This project analyzes **student feedback survey data** to evaluate satisfaction levels across different courses. The analysis calculates weighted average ratings, classifies sentiment, and visualizes trends to help academic teams understand learner experience and improve course quality.

---

##  Features Implemented

* Data cleaning and column normalization
* Calculation of:

  * Total responses
  * Weighted average course rating
* Satisfaction level labeling:

  * Highly Satisfied
  * Satisfied
  * Neutral
  * Unsatisfied
* Sentiment classification:

  * Positive
  * Neutral
  * Negative
